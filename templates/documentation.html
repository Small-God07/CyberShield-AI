Présentation du projet

CyberShield AI est un système de détection et de résolution automatique de cyberattaques basé sur l'Intelligence Artificielle (Random Forest), développé avec Flask (Python). Il analyse les métriques réseau en temps réel pour identifier six types de menaces et proposer des actions de remédiation adaptées.

CyberShield AI is an AI-powered cybersecurity system built with Flask and scikit-learn. It uses a Random Forest classifier trained on simulated network connection data to detect six types of cyber threats and automatically recommend remediation actions.

Le Random Forest est un ensemble d'arbres de décision entraînés sur des sous-ensembles aléatoires des données. Il est robuste au sur-apprentissage et fournit une estimation de confiance via les probabilités moyennées sur tous les arbres.

Variables d'entrée (features)
  1. duree_connexion — Durée de la connexion (secondes)
  2. nb_connexions_par_sec — Nombre de connexions par seconde
  3. volume_donnees_kb — Volume de données échangées (KB)
  4. nb_erreurs — Nombre d'erreurs réseau
  5. nb_ports_scanes — Nombre de ports scannés
  6. taux_echec_connexion — Taux d'échec des connexions (0–1)
  7. taille_paquets_moy — Taille moyenne des paquets (bytes)
  8. nb_tentatives_auth — Nombre de tentatives d'authentification

Glossaire
DDoS — Distributed Denial of Service : saturation d'un service par un flux massif de requêtes.
Intrusion — Accès non autorisé à un système via exploitation de vulnérabilités.
Malware — Logiciel malveillant (virus, trojan, ransomware…).
Phishing — Hameçonnage : tentative de vol d'identifiants par imitation.
SQL Injection — Injection de code SQL malveillant dans une requête.
Random Forest — Ensemble d'arbres de décision pour la classification.
WAF — Web Application Firewall : pare-feu applicatif web.
Rate Limiting — Limitation du nombre de requêtes par unité de temps.
